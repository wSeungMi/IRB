# 회고11조 스터디모임 '일리뿌' 공부기록

### 스터디소개

일케(함께 공부)해서 리액트까지 뿌시자 스터디입니다.  
코드 아웃풋 연습 및 개발언어 공부 습관을 바꾸기위해 시작한 스터디로 함께 공부하며 자신에게 맞는 공부습관을 찾는데 의의를 두고 있습니다.

진행기간 : 2022년 10월 24일~ 프로젝트 기간 전까지  
진행날짜 : 매주 평일 오후 4시~6시(자습시간 활용)  
참여인원 : 권지혜, 김현빈, 우승미
